# Delta Exchange India Dashboard A mobile-first web application for monitoring trading activity on Delta Exchange with AI-powered analysis. ## Features - Portfolio overview with balance and margin information - Open positions monitoring with leverage warnings - Trade history with fees and P&L tracking - Market data for popular instruments - AI-powered trading performance analysis - Mobile-first responsive design ## Setup 1. Install dependencies: npm install 2. Configure environment variables in .env.local 3. Run the development server: npm run dev 4. Open http://localhost:3000 ## Environment Variables - DELTA_API_KEY: Your Delta Exchange API key - DELTA_API_SECRET: Your Delta Exchange API secret - OPENAI_API_KEY: Your OpenAI API key ## API Routes - /api/delta/trades: Fetches trading data from Delta Exchange - /api/openai/analyze: Analyzes trading performance with AI ## Security All API calls are handled server-side to keep keys secure. ## Tech Stack - Next.js 14 - TypeScript - Tailwind CSS - Lucide React Icons - OpenAI GPT-4
